# 🧠 AI Skill: bgustdown

**High-performance data engineering tool for AI Agents.**

This skill provides native capabilities for AI agents (Gemini, Claude, GPT) to process local documents with extreme speed and precision.

---

## 🇪🇸 Español

### Instalación de la Skill
Para integrar esta herramienta en tu agente IA de forma estandarizada:

```bash
npx skill add https://github.com/B-GUST/bgustdown
```

### Comandos Disponibles
| Comando | Acción |
| :--- | :--- |
| `npx bgustdown convert <file>` | Convierte documento a Markdown limpio. |
| `npx bgustdown prepare <file>` | Dataset JSON segmentado para NLP. |
